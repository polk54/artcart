# ArtCart — landing page

The public "coming soon" page for **ArtCart**, a calm, inventory-first tracker
for craft projects (iOS app in progress).

Static site served via GitHub Pages → https://polk54.github.io/artcart

- `index.html` — the page (self-contained; dot-grid background, wordmark, Hanken Grotesk)
- `artcart-logo.png` — transparent wordmark
- `icon.png` — app icon (favicon + social share)

The iOS app source lives in a separate private repo.
